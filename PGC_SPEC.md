Scholars Access

* Log in credentials  
* Able to access online or offline  
* Able to view and update the following. All edit or update will proceed for review and approval of evaluator before it syncs to the system. Before they submit their updated data there must be a prompt question before they proceed.   
1. Scholars Profile – Batch Number, Year level & Course and other details from application form (Personal Data, Educational Background  
2. Academic Performance  
* Regular or irregular student – Able to update   
* Able to update their grades for every school year by semester. They can update their prospectus by inputing their grades.   
* They can select or add on the remarks for their grades status if it is complete, (2) unposted, (1) INC or have (1) failed grade for every semester.   
* Able to view their status from the history of their program they shift  
3. Along with grades they must upload the following requirements for checking of evaluator

* Certificate of Registration (COR)

* Report of Grades (ROG)

* Explanation Letter (optional)

* Completion Form (optional)

* Additional documents (optional)  
4. Able to view the status update:   
* Active/ Inactive or Graduate  
5. Able to view the events attended in CSR Events  
6. Able to view the record of allowance received. 

Evaluator Access

1. Log in Credentials  
2. Able to view and update the scholars’ profile  
3. Able to generate the following report  
* Number of Active Scholars by school and by batch  
* Number of Inactive Scholars by school & by batch   
* Number of Graduate Scholars by school & by batch  
* Number of extend – active scholars by school & by batch  
* Number of Latin Honors – Summa, Magna, Cum Laude by school & by batch  
* Post-graduation reports   
* Number of scholars currently employed @ PPS  
* Number of Licensure Passers per Profession  
4. Able to update the post-graduation status  
* If employed @ PPS  
* Board Passer and date of passing the board exam  
5. Filter the data by course, school, batch, status & demographics  
6. Able to update the allowance received by the scholars  
7. Able to update the status in CSR Participation

### **Write/Add Evaluation Remarks**

Evaluators may:

* Add written or predefined remarks (e.g., First Warning Notice)  
* Track the history of remarks issued  
* Monitor missing uploads and incomplete requirements  
* Record additional notes for faster evaluation and follow-up
